## 🔧 Como substituir os placeholders

Cada placeholder no código segue o padrão:

```html
<!-- Placeholder atual -->
<div class="img-placeholder">
  <div class="ph-gradient"></div>
  <div class="ph-grain"></div>
  <div class="ph-label">hero-main.jpg</div>
</div>

<!-- Substituir por: -->
<img src="assets/images/hero-main.jpg" alt="..." class="img-real">
```

E nos slides (`slides.html`), o padrão é:

```html
<!-- Placeholder atual -->
<div class="s-img s-img-banner">
  <div class="ph-grad"></div>
  <div class="ph-grain"></div>
  <div class="ph-label">hero-main.jpg</div>
</div>

<!-- Substituir por: -->
<div class="s-img s-img-banner">
  <img src="assets/images/hero-main.jpg" alt="..." style="width:100%;height:100%;object-fit:cover;">
</div>
```

O label `ph-label` serve de guia para localizar onde vai cada arquivo.
